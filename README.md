## Bash Autoscroll                                                                                                  
A simple pure bash autoscroll                                                                                       
                                                                                                                    
### Requirements                                                                                                    
* xdotool                                                                                                           
* xbindkeys (Optional)                                                                                              
                                                                                                                    
### Usage                                                                                                           
* Make the scripts executable   
`chmod +x auto_scroll`   
`chmod +x auto_scroll_stop`
